# **App Name**: Blood Bridge

## Core Features:

- Splash Screen: Animated splash screen with pulsing blood drop icon, app name, and tagline.
- Welcome Page: Landing page with background video, tagline, and buttons for dashboard access and about us information.
- Resource Inventory Dashboard: Display critical/low supply counts, filters, and resource cards with blood type, quantity, location, status, and update button.
- Urgent Request Broadcasting: Form for hospitals to submit urgent blood requests with blood type, quantity, urgency, hospital details, and broadcast radius.
- Active Requests Display: Show a grid of active blood requests, and a function where donors can virtually claim the intent to donate. Use LLM tool to filter which hospitals the donor should be referred to first.
- Donation Camp Management: Interactive map and grid to display upcoming blood donation camps, including registration with personal details, photo capture, and participation ticket generation.
- Data Analytics Visualization: Charts to display blood inventory by type and status for analytics purposes.

## Style Guidelines:

- Primary color: Deep crimson (#8B0000), symbolizing the life-giving nature of blood and evoking a sense of urgency.
- Background color: Light gray (#F0F0F0) for a clean and professional feel, allowing the crimson to stand out.
- Accent color: Soft blue (#ADD8E6), an analogous color to crimson that is calming and inspires trust.
- Body text: 'PT Sans' (humanist sans-serif) for readability and a modern yet approachable aesthetic.
- Headline Font: 'Space Grotesk' (sans-serif) which gives the site a slightly modern and techy appearance.
- Utilize the 'Lucide React' icon set for clear and consistent icons throughout the application.
- Implement a collapsible sidebar and header bar for navigation.
- Subtle animations for the splash screen and transitions.
